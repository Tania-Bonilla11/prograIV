    SOBRE NOSOTROS...

    Somos una aplicación web comprometida con la seguridad vial de los salvadoreños ofreciendoles una platorma interactiva con juegos y videos para un aprendizaje vial efectivo,también prestamos servicio a la división de la PNC terrestre de educación vial,siendo para ellos una herramienta de registro de capacitaciones para charlas en instituciones

    Al Registrarte e Iniciar Sesion puedes encontrar:

    Juegos para ejercitar la memoria

    Contamos con juegos para un aprendizaje divertido,para niños y adultos,la educación vial es para todas las edades

    Graficas de accidentes en el Pais

    Al iniciar sesion podras observar las graficas de accidentes de transito y sus resspectivas causas

    Simulacion de Test para obtener licencia de Conducir

    Con mi guia vial el salvador puedes realizar simulaciones de los test para obtener la licencia de conducir ya sea el test psicologico,test de señalizacion,vvmt entre otras dandote un resultado obtenido de la prueba de manera automatica,preparandote para el test real
